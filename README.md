## Description

Using NestJs framework and RestAPI, TypeORM, PostgreSQL and Redis.
Please make sure you have already install redis and PostgreSQL in your local machine or container

## Add information or configure your env file from the env folder

Please give the all required information

- development.env
- staging.evn
- production.env

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# staging mode
$ yarn run start:staging


# build
$ yarn run start:build

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
